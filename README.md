MEAN-Todo-App
=============

A Todo MEAN App - using MongoDB, Expressjs, Angularjs, Nodejs, REST API, JSON.

### Install
* Download/clone the repo
* Run `npm install`
* Run `gulp` and navigate to `http://localhost:3000` to view the app
